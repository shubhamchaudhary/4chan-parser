# 4chan-parser

## Usage: 4chan-parser [options]

###Options:

| Option    | Alternative       | Explanation                     |
| :---------| :-----------------| :-------------------------------|
| -h,       | --help            | Show this help message and exit |
| -t THREAD,| --thread=THREAD   | Thread number |
| -b BOARD, | --board=BOARD     | Board name |
